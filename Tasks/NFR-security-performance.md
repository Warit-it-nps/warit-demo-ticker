# Task: NFR ความปลอดภัยและประสิทธิภาพระบบ

## เป้าหมาย
ตรวจสอบและรับรองว่าระบบปฏิบัติตามข้อกำหนดที่ไม่ใช่ฟังก์ชันใน PRD หมวด 7

## รายละเอียดงาน
- บังคับใช้ HTTPS และนโยบายการเข้ารหัสข้อมูลสำคัญ (เช่น Token, ไฟล์แนบ)
- ตรวจสอบสิทธิ์การเข้าถึง Ticket ให้ User เห็นเฉพาะของตนเอง
- ตั้งค่าเครื่องมือวัด Performance (Lighthouse/Monitoring) ให้หน้าโหลดภายใน 3 วินาที
- ออกแบบ UX ให้ผู้ใช้สร้าง Ticket ใหม่ได้ภายใน 3 คลิกจากหน้าแรก
- ทดสอบ Compatibility กับ Chrome, Firefox, Edge เวอร์ชันล่าสุด

## เงื่อนไขสำเร็จ
- รายงานการทดสอบความปลอดภัยและ performance ถูกจัดเก็บเป็นหลักฐาน
- ปัญหาที่พบในการทดสอบถูกแก้ไขหรือมีแผนรองรับ
- มี checklist สำหรับ regression test ด้าน NFR ใน sprint ต่อๆ ไป

## อ้างอิง PRD
- หมวด 7: ข้อกำหนดที่ไม่ใช่ฟังก์ชัน
- หมวด 2: ปัญหาเรื่องความปลอดภัยและความโปร่งใส
