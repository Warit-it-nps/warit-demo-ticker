# Task: US-5.1 คิวงาน Ticket สำหรับ Admin

## เป้าหมาย
ให้ทีม IT ดูรายการ Ticket ทั้งหมดและกรองตามเกณฑ์สำคัญได้สะดวก

## รายละเอียดงาน
- พัฒนาหน้าคิว Ticket ที่รวมข้อมูลสำคัญเช่น ผู้แจ้ง สถานะ Priority เวลาอัปเดตล่าสุด
- เพิ่มฟิลเตอร์ตามสถานะ Priority ผู้รับผิดชอบ และประเภทปัญหา
- รองรับการค้นหาด้วยข้อความ
- เชื่อมต่อ API เพื่อดึงรายการพร้อม pagination และ sorting

## เงื่อนไขสำเร็จ
- Admin สามารถกรอง Ticket ตามเงื่อนไขได้พร้อมกันหลายตัว
- ตอบสนองรวดเร็วภายใต้โหลดปกติ (ไม่เกิน 3 วินาที)
- ปัจจุบันผู้ใช้ทั่วไปไม่สามารถเข้าถึงหน้านี้ได้
