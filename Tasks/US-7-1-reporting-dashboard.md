# Task: US-7.1 รายงานสำหรับ IT Manager

## เป้าหมาย
ให้ผู้จัดการ IT ดูรายงานสรุปเพื่อวัดผลทีมและวางแผนล่วงหน้า

## รายละเอียดงาน
- สร้างหน้ารายงานแสดงกราฟจำนวน Ticket รายเดือน, ประเภทปัญหายอดนิยม, เวลาเฉลี่ยในการแก้ไข
- รองรับการเลือกช่วงเวลาและการส่งออก (CSV/PDF) หากจำเป็น
- เพิ่มตัวชี้วัดหลัก เช่น Adoption Rate, CSAT ในรูปแบบการ์ด
- ตรวจสอบสิทธิ์ให้เฉพาะ Manager หรือ Admin ที่ได้รับอนุญาตเข้าถึงได้

## เงื่อนไขสำเร็จ
- ข้อมูลรายงานดึงจากแหล่งข้อมูลเดียวกันกับระบบหลัก (Single Source of Truth)
- ผู้ใช้สามารถกรองช่วงเวลาหรือหน่วยงานได้และรายงานอัปเดตทันที
- รองรับการแสดงผลบนเบราว์เซอร์หลักทั้งหมดตามข้อกำหนด Compatibility
