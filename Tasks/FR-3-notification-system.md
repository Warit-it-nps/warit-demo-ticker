# Task: FR-3 ระบบแจ้งเตือนอัตโนมัติ

## เป้าหมาย
สร้างโครงสร้างการแจ้งเตือนที่รองรับทุกเหตุการณ์ตามข้อกำหนด FR-3

## รายละเอียดงาน
- ออกแบบ service กลางสำหรับส่งอีเมลและรองรับการต่อยอดเป็น in-app notification
- กำหนดเทมเพลตอีเมลสำหรับเหตุการณ์: Ticket ใหม่, Admin ตอบกลับ, ปิดงาน, User ตอบกลับ
- สร้างคิวหรือกลไก retry หากส่งล้มเหลว
- จัดทำ log สำหรับตรวจสอบย้อนหลัง

## เงื่อนไขสำเร็จ
- ทุกเหตุการณ์ตาม FR-3 ส่งอีเมลได้สำเร็จและทดสอบครอบคลุม
- มี monitoring หรือ alert เมื่อเกิด backlog หรือ error
- โครงสร้างรองรับการเพิ่มช่องทางแจ้งเตือนในอนาคต
