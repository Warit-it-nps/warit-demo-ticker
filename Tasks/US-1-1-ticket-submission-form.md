# Task: US-1.1 แบบฟอร์มแจ้งปัญหา

## เป้าหมาย
เปิดให้ User กรอกแบบฟอร์มที่ใช้งานง่ายและครบถ้วนสำหรับสร้าง Ticket ใหม่

## รายละเอียดงาน
- ออกแบบ UI/UX สำหรับฟอร์มพร้อมคำอธิบายสั้นในแต่ละช่อง
- รองรับการเลือกประเภทปัญหา หัวข้อ รายละเอียด และการแนบไฟล์ (สูงสุด 10MB)
- สร้าง Ticket ID ไม่ซ้ำกันเมื่อกดส่ง
- ตรวจสอบ validation ฝั่ง client และ server

## เงื่อนไขสำเร็จ
- User สร้าง Ticket ได้สำเร็จพร้อมแนบไฟล์ตามข้อกำหนด
- Ticket ID ถูกสร้างตามรูปแบบองค์กร (เช่น IT-YYYY-XXXX)
- ฟอร์มแจ้งข้อผิดพลาดชัดเจนเมื่อกรอกไม่ครบหรือไฟล์ไม่ถูกต้อง
