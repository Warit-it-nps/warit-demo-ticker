# Task: US-5.2 การรับและส่งต่อ Ticket

## เป้าหมาย
ให้ Admin รับงานหรือส่งต่อ Ticket ไปยังสมาชิก IT คนอื่นได้สะดวก

## รายละเอียดงาน
- เพิ่มปุ่ม Assign/Unassign ในหน้าคิวและหน้ารายละเอียด
- รองรับการเลือกผู้รับงานจากรายชื่อทีม IT (ค้นหา/กรองได้)
- บันทึกประวัติการเปลี่ยนผู้รับงานใน timeline ของ Ticket
- แจ้งเตือนผู้รับงานใหม่ผ่านอีเมล

## เงื่อนไขสำเร็จ
- Admin สามารถรับงานและส่งต่อได้ในไม่กี่คลิก
- ประวัติการมอบหมายแสดงชัดเจนพร้อมเวลาและผู้ดำเนินการ
- แจ้งเตือนถูกส่งถึงผู้รับใหม่ทุกครั้ง
