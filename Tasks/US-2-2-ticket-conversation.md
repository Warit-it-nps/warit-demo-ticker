# Task: US-2.2 ช่องทางโต้ตอบใน Ticket

## เป้าหมาย
เปิดให้ User และ Admin สื่อสารโต้ตอบกันภายในหน้ารายละเอียด Ticket

## รายละเอียดงาน
- ออกแบบส่วนแสดงประวัติการสื่อสารแบบเรียงตามเวลา
- รองรับการแนบไฟล์/รูปในการตอบกลับ (ถ้าจำเป็นให้ reuse validation จากฟอร์ม)
- แจ้งเตือนคู่สนทนาด้วยอีเมลหรือ in-app notification เมื่อตอบกลับสำเร็จ
- ป้องกันการเห็นข้อความของผู้อื่นที่ไม่เกี่ยวข้องด้วยการตรวจสิทธิ์

## เงื่อนไขสำเร็จ
- User และ Admin สามารถโพสต์ข้อความและเห็นข้อความของอีกฝ่ายได้ทันที
- ระบบ timestamp ชัดเจนและเรียงถูกต้อง
- การแนบไฟล์ทำงานได้และแสดงตัวอย่าง/ลิงก์ดาวน์โหลด
