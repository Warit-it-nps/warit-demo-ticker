# Task: US-6.1 จัดการบทความคลังความรู้

## เป้าหมาย
ให้ทีม IT สร้าง แก้ไข และลบบทความ Knowledge Base ได้สะดวก

## รายละเอียดงาน
- พัฒนาหน้าจัดการบทความพร้อมตารางรายการและตัวกรอง
- สร้างฟอร์มแก้ไข/สร้างบทความรองรับ rich text และการแนบไฟล์/รูป
- จัดการสิทธิ์ให้เฉพาะ Role Admin เท่านั้นที่เข้าถึงส่วนนี้ได้
- บันทึกประวัติการเปลี่ยนแปลง (versioning หรือ draft/publish หากจำเป็น)

## เงื่อนไขสำเร็จ
- Admin สามารถสร้าง แก้ไข ลบ และเผยแพร่บทความได้ครบวงจร
- การเปลี่ยนสถานะบทความสะท้อนในหน้า Knowledge Base ของ User
- ระบบเก็บบันทึกว่าผู้ใดเป็นคนแก้ไขล่าสุด
