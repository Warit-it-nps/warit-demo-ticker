# Task: US-5.4 บันทึกข้อความภายในทีม IT

## เป้าหมาย
อนุญาตให้ทีม IT เขียนบันทึกที่ User ไม่เห็น เพื่อการประสานงานภายใน

## รายละเอียดงาน
- เพิ่มช่อง Internal Note ในหน้ารายละเอียด Ticket พร้อมระบุผู้บันทึก
- แยกการแสดงผลระหว่าง internal notes กับ public replies อย่างชัดเจน
- ป้องกันการมองเห็นสำหรับ Role User
- เก็บ log การแก้ไข/ลบ internal note สำหรับตรวจสอบย้อนหลัง

## เงื่อนไขสำเร็จ
- เฉพาะ Admin เท่านั้นที่สร้าง/ดู/แก้ไข internal note ได้
- Internal note ไม่ถูกส่งไปยังผู้ใช้ทางอีเมล
- ประวัติการบันทึกแสดงตามเวลาและผู้เขียนครบถ้วน
