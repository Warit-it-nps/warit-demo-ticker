# Task: US-4.1 แดชบอร์ดผู้ดูแลระบบ

## เป้าหมาย
ให้ Admin เห็นสถานการณ์ภาพรวมของ Ticket ทันทีหลังล็อกอิน

## รายละเอียดงาน
- สร้างแดชบอร์ดแสดงจำนวน Ticket ใหม่, ค้าง, ปิดแล้ว แบบ real-time
- เพิ่มกราฟหรือการ์ดแสดงงานที่ต้องเร่งด่วน (เช่น Priority สูง, Ticket เกิน SLA)
- รวมปุ่มลัดไปยังงานค้างหรือฟิลเตอร์สำคัญ
- ตรวจสอบสิทธิ์ให้เฉพาะ Admin เข้าถึงได้

## เงื่อนไขสำเร็จ
- ข้อมูลแดชบอร์ดอัปเดตเมื่อมีการเปลี่ยนแปลง Ticket
- โหลดหน้าไม่เกิน 3 วินาทีภายใต้ปริมาณข้อมูลปกติ
- รองรับการแสดงผลบนจอ desktop ที่ความละเอียดขั้นต่ำ 1280px
