# Task: US-1.2 อีเมลยืนยันการแจ้งปัญหา

## เป้าหมาย
ส่งอีเมลยืนยันไปยัง User หลังสร้าง Ticket เพื่ออ้างอิงและยืนยันการรับเรื่อง

## รายละเอียดงาน
- ออกแบบเทมเพลตอีเมล (รองรับเนื้อหาไทย/อังกฤษเบื้องต้น)
- เรียกกระบวนการส่งอีเมลเมื่อ Ticket ถูกสร้างสำเร็จ
- ใส่ข้อมูล Ticket ID, สถานะเริ่มต้น, ลิงก์กลับไปยังรายละเอียด Ticket
- บันทึกสถานะการส่งเพื่อการตรวจสอบภายหลัง

## เงื่อนไขสำเร็จ
- User ได้รับอีเมลทันทีหลังสร้าง Ticket
- หากส่งอีเมลไม่สำเร็จ ระบบต้องล็อกและแจ้งเตือน Admin
- เทมเพลตต้องแสดงผลถูกต้องบนอีเมลหลัก (Outlook, Gmail, Apple Mail)
