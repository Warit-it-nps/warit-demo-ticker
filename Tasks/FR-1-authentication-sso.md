# Task: FR-1 การยืนยันตัวตนและสิทธิ์ผู้ใช้

## เป้าหมาย
เชื่อมต่อระบบกับ Active Directory (SSO) และกำหนด Role ระหว่าง User กับ Admin ตามข้อกำหนด FR-1

## รายละเอียดงาน
- ตั้งค่าการเชื่อมต่อ AD/LDAP สำหรับ Single Sign-On
- ออกแบบโครงสร้าง Role-Based Access Control เพื่อแยกระหว่าง User และ Admin
- กำหนดการป้องกันการเข้าถึงหน้า/ข้อมูลเฉพาะ Role ทั้งฝั่ง UI และ API

## เงื่อนไขสำเร็จ
- ผู้ใช้ใน AD ต้องล็อกอินได้โดยไม่ต้องสร้างบัญชีใหม่
- หลัง SSO ระบบต้องรู้บริบท role ของผู้ใช้ทันที
- ผู้ใช้ทั่วไปไม่สามารถเข้าถึงฟังก์ชัน Admin ได้ และกลับกัน
