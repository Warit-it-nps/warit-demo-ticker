# Task: US-2.1 รายการ Ticket ของผู้ใช้งาน

## เป้าหมาย
ให้ User ดูรายการ Ticket ที่ตนเคยแจ้งพร้อมสถานะปัจจุบันได้แบบเรียลไทม์

## รายละเอียดงาน
- สร้างหน้ารวม Ticket ที่แสดงสถานะ, วันที่สร้าง, Priority แบบย่อ
- เพิ่มตัวกรองพื้นฐาน (เช่น สถานะ, ช่วงวันที่)
- รองรับการกดเข้าไปดูรายละเอียดแต่ละ Ticket
- ดึงข้อมูลจาก API พร้อม pagination

## เงื่อนไขสำเร็จ
- User เห็น Ticket ของตัวเองเท่านั้น
- หน้ารายการโหลดภายใน 3 วินาทีตาม NFR
- ค่า default เรียงตามวันที่อัปเดตล่าสุด
